# カスタムカードセット設計

## 概要

ルーム作成時に、既存の3プリセット（フィボナッチ / Tシャツ / 2のべき乗）に加え、「カスタム」オプションを追加。ユーザーが数値カードを自由に定義できる。

## UI

- ルーム作成フォームのカードセット選択に **4つ目のラジオボタン「カスタム」** を追加
- 「カスタム」選択時にテキスト入力欄が表示される
- 入力形式: カンマ区切りの数値（例: `0.5, 1, 2, 3, 5, 8`）
- バリデーション: 最低2枚、最大20枚、数値のみ

## カード構成

- カスタム選択時: **入力した数値カード + 特殊カード（?, ∞, ☕）** のみ表示
- 既存プリセットへの追加ではなく、数値部分を完全に置き換える

## データ

- `rooms.card_set` カラムに `custom` 値を追加
- 新規カラム `rooms.custom_cards`（`TEXT[]` 配列）でカスタム値を保存
- 既存プリセット選択時は `custom_cards` は NULL

## 統計計算

- カスタム数値カードは統計（平均・中央値・最頻値）の計算対象
- 特殊カード（`?`, `∞`, `☕`）は引き続き除外
