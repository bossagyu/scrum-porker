# i18n（国際化）設計

## 概要

日本語と英語の2言語に対応。UIトグルで切り替え、URLは変更しない。

## 方針

- **ライブラリ**: next-intl（ルーティングなしモード）
- **言語切替**: ヘッダーに切替ボタン、localStorage で保持
- **デフォルト言語**: ブラウザの `navigator.language` で自動判定（ja → 日本語、それ以外 → 英語）
- **スコープ**: 個人ごとに独立（ルーム単位ではない）

## 翻訳ファイル

- `messages/ja.json` / `messages/en.json`
- 名前空間: common, createRoom, joinRoom, voting, cardSets, timer, participant, roomSettings, history, statistics, roomView, errors, validation, metadata

## Server Actions のエラーメッセージ

- Server Actions はエラーキー（`errors.auth` 等）を返す
- クライアント側で `useTranslations` を使って翻訳表示

## 対象ファイル

- コンポーネント: 10ファイル
- ページ: 6ファイル
- アクション: 3ファイル
- ユーティリティ: 2ファイル（constants.ts, export-utils.ts）
- 合計: 約90文字列
